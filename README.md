# Signup Form

A simple, clean Signup page built with HTML and CSS.

## 🔗 Live Demo

live link: https://aj1234-p.github.io/SignUp/

## 📋 Features

- Full Name, Email, Password, and Confirm Password fields
- Date of Birth picker
- Radio buttons to select **Student** or **Employee**
- "I agree to terms & conditions" checkbox
- Required field validation using native HTML5 `required` attributes
- Sleek **dark theme** UI (black background, white text, minimal underline-style inputs)
- Custom favicon (multi-size + Apple touch icon + web manifest)

## 🚀 Getting Started

### Run locally

1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   ```
2. Open the project folder:
   ```bash
   cd <repo-name>
   ```
3. Open `index.html` directly in your browser, or use a live server (e.g. VS Code's "Live Server" extension) for auto-reload while editing.
